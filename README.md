# Smart Study Time Analyzer

## Description
Smart Study Time Analyzer is a simple Python console-based application that helps analyze weekly study habits. The program collects study hours for each day of the week and provides useful insights such as total study time, average study hours, and the most productive study day.

## Features
- Takes study hours for 7 days as input
- Calculates total weekly study hours
- Calculates average study hours per day
- Identifies the most productive day
- Provides basic feedback on study consistency

## Technologies Used
- Python

## How It Works
1. The user enters study hours for each day of the week.
2. The program stores the data in a list.
3. It analyzes the data to compute total and average study time.
4. It finds the day with the maximum study hours.
5. A feedback message is displayed based on the average study hours.

python smart_study_time_analyzer.py
